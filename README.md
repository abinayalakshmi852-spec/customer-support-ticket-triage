Customer Support Ticket Automation

This project automates the analysis of customer support tickets using Python. It cleans ticket text, classifies issues, assigns priority levels, calculates SLA deadlines, and generates a final structured report. The system helps support teams quickly understand ticket urgency and issue types, improving response efficiency.

Tech Stack: Python, Pandas
Input: Customer support ticket CSV file
Output: Processed CSV with issue category, priority, SLA hours, and due time
